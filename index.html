The following major projects I buid :

Crime Reporting System

The Crime Reporting System is a web-based application designed to simplify and digitalize the process of reporting crimes. In many areas, reporting a crime can be time-consuming, inconvenient, or intimidating due to manual paperwork and long waiting times at police stations. This project aims to solve these issues by providing a secure and user-friendly platform where citizens can report criminal activities online.

Co-Worker helpline Service

The Co-Worker Helpline Service is a web-based platform developed to help employees or workers in an organization connect with HR or support staff to report workplace issues, seek help, or raise concerns in a confidential and organized manner. This system ensures smooth internal communication, improves workplace safety, and promotes a healthy work environment.

Rain Drop Sensor

The Rain Drop Sensor Project is an IoT-based automatic rain detection system designed using Arduino Uno, a Rain Drop Sensor, and a Servo Motor. The main aim of this project is to detect the presence of rain and take automatic action, such as closing a window, roof, or covering a device, using a servo motor. This system is useful in smart agriculture, smart homes, or any weather-sensitive applications.
